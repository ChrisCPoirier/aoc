### Testing
`go test -timeout 10s -run ^TestParts$ aoc/cmd/year[year]/day[day]`

example: `go test -timeout 10s -run ^TestParts$ aoc/cmd/year2024/day7`

### Execute 

`go run main.go [year] day[day]`

example: `go run main.go 2024 day1`
